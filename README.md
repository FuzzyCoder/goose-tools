# goose-tools

Canonical portable agent toolchain for the [Goose](https://github.com/block/goose) platform.

> **Status:** In progress — migrating from warp-tools.

Provides a structured **Plan → Review → Edit → Approve → Finalize → Execute** workflow,
a reusable code-quality kit (13-rule slop linter + remediation skills), and a portable
skill library — all built for the Goose platform.
